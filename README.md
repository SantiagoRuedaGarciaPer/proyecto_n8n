# Proyecto de automatización de n8n
Este proyecto fue diseñado para la automatización de validación de ausencias en una escuela a tra vez de una IA

## Instalación
### Archivos
Descarga el archivo [n8n.json](./n8n.json) e importalo en un workflow de n8n
Descarga igualmente los archivos en la carpeta [front](./front), donde se encuentra el html, css y javascript del formulario de inasistencia.

### Relaciones
Ingresa al siguiente documento de [google sheets](https://docs.google.com/spreadsheets/d/1ez2GoQdzKL5hrRNKEhoRiOh_7rtOO7kQXsJ2Rjq0tNM/edit?gid=0#gid=0) donde podras ver el formato del documento y copiarlo a uno de tu autoria
Crea un bot de telegram y copia su acces token para ingresarlo luego

## Credenciales(desde la pagina web de n8n, para hacerlo de forma local investiga en google como hacerlo)
- Para google sheets inicia secion normal con google
- Para telegram, ingresa el accses token del bot que creaste anteriormente
- Para gmail, usa el mismo procedimiento que con google sheets
